# The Education-Premium Paradox: Immigrant Mortgage Underwriting
This repository contains the Python source code and Tableau visualization files for my Master's thesis on the Human Capital Wealth Capacity Index (HC-WCI). 

- **Python Scripts:** Contains the raw data extraction pipeline and the Random Forest machine learning model.
- **Tableau Visualizations:** The `.twbx` file contains the geospatial heat maps and longitudinal velocity curves.
